# 📷 博客图片展示与缩放测试

这篇笔记用于验证站点内图片资源的展示效果，尤其是正文图片在桌面端和移动端的宽度、缩放、留白与点击预览体验。

## 检查要点

- 图片是否能正常加载。
- 图片宽度是否跟随正文容器自适应。
- 点击图片后，是否能触发放大预览。
- 深色模式下，图片周围留白是否自然。

如果后续调整了 VitePress 主题、`medium-zoom` 配置或图片样式，可以回到这篇文章快速确认显示是否仍然正常。
